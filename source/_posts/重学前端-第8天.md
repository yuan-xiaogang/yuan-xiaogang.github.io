2、日期对象

只能通过date的构造函数创建日期对象；

|方法|说明|
|---|---|
|getFullYear()|获取当年|
|getMonth()|获取当月，返回的月份小一个月；+1|
|getDate()|获取当天日期|
|getDay()|获取星期几（周日0-周六6）|
|getHours()|获取当前小时|
|getMinutes()|获取当前分钟|
|getSeconds()|获取当前秒钟|

起始时间1970年一月一号；
获取毫秒数： valueOf() getTime();

`var date1 = +new Date()` 返回的是毫秒数
`Date.now()` h5新增方法；返回的是毫秒数

3、数组对象
检测是否为数组：
1、instanceof
`arr instanceof Array`
2、Array.isArray() h5新增的，ie9以上
`Array.isArray(arr)`

|方法|说明|返回值|
|---|---|---|
|push|往数组后面添加一个或者多个元素，原数组也发生变化|返回新长度|
|pop|删除最后一个元素，只能是一个|返回删除的元素|
|unshift|向数组的开头处添加一个或者多个元素|返回新长度|
|shift|删除数组的开头元素|返回删除的元素|
|reverse|反转数组|返回新数组|
|sort|冒泡排序，个位数可以|返回新数  组|
```
arr.sort(function (a,b) {
    return a-b // 升序
    return b-a // 降序
})
```

获取索引号
indexOf(); 数组中的第一个索引号
lastIndexOf(); 数组中的最后一个索引号

转化成字符串

join('')
